Overview

Vers Suite adalah aplikasi proxy intersepsi untuk analisis HTTP/HTTPS, request replay, dan payload fuzzing. Cocok untuk pembelajaran keamanan aplikasi web dan penetration testing berizin.

Core Stack

  • Python 3.10+
  • PyQt5 desktop UI
  • mitmproxy engine

Main Modules

  • Intercept + HTTP History
  • Repeater
  • Intruder
  • Decoder

Use Case

  • Inspect requests/responses
  • Header & WAF detection
  • Manual replay and fuzzing

Features

Intercept Proxy

Forward, drop, forward all, drop all, dan editing request secara realtime.

HTTP History

Pencatatan traffic lengkap dengan context menu dan kirim cepat ke Repeater/Intruder.

Advanced Metadata

Deteksi IP target, indikasi WAF, Cloudflare, server signature, dan security headers.

Repeater

Replay request manual, grouping/ungrouping session, serta search/filter host-path-group.

Intruder

Pengujian payload multithread untuk fuzzing endpoint.

HTTPS Support

Panduan sertifikat CA untuk Chrome, Edge, Brave, Opera, Firefox, dan Safari (macOS).

Installation

  1. Clone repository Vers Suite.
  2. Buat virtual environment.
  3. Install dependency dari requirements.txt.
  4. Jalankan aplikasi dengan python main.py.
git clone https://github.com/Xnuvers007/verssuite.git
cd verssuite
python -m venv venv
# Windows
venv\Scripts\activate
pip install -r requirements.txt
python main.py

Quick Usage

  1. Set browser proxy ke 127.0.0.1:PORT (PORT harus sama dengan yang Anda pilih di Vers Suite).
  2. Start proxy dari header bar.
  3. Aktifkan Intercept jika ingin hold request.
  4. Kirim request dari History ke Repeater/Intruder sesuai kebutuhan testing.
  5. Buka tombol SSL Cert untuk setup sertifikat HTTPS.

FAQ

PORT harus selalu 8080?

Tidak. Anda bebas memilih port di Vers Suite. Browser/proxy client cukup mengikuti port yang sama.

Kenapa HTTPS belum terbaca?

Pastikan CA certificate sudah terpasang & trusted, lalu restart browser dan pastikan proxy aktif.

Apakah tool ini untuk aktivitas ilegal?

Tidak. Gunakan hanya untuk edukasi dan pengujian keamanan pada sistem yang memiliki izin resmi.
